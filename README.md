# PHP_pizzaria

## Criando um projeto com PHP &amp; MYSQL, Pizzaria Online.

## Neste projeto desenvolvido no curso MYSQL do básico ao avançado,
## utilizei a linguagem PHP, HTML linguagem de marcação & CSS para estilização deixando assim o site totalmente responsivo, e conectado com o nosso Banco de Dados MYSQL.
